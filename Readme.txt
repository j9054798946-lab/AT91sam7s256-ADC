//В этой директории изначально был проект мигалка "Вильгельм Тель" светодиодом: "та-да-дам, та-да-дам, та-да-дам, там, там". Ритм задается одной строкой по принципу: нота = два параметра {1,2} , где 1 = длительность свечения СВД, 2 = длительность гашения СВД. Длительности вырабатываются с помощью прерываний от таймера  TC0. Это был первый проект составленный на основе battle Gemini - GPT на сайте Lmarena, что и дало название проекту. Но потом в режиме only GPT работа была продолжена  и все файлы были модифицированы под содержимое проекта Led_to_PC - передача статуса светодиода - при этом название файлов остались прежними, включая название проекта. 
************************************
Таким образом, здесь оказался проект прошивки для At91sam7s256 "красный кружок" (Led_to_PC) с "неправильным" названием LM_gemini_gpt с функционалом: отправка статуса светодиода по TCP с платы, имеющей локальный адрес 192.168.0.7:23,  на ПК  поочередно символа единицы ( код 0x31 ) --> красный цвет кружка и символа нуля ( код 0x30 ) --> белый цвет кружка. 
************************************
Следующим шагом в проекте прошивки  должно стать считывание АЦП1 ( имеющегося на плате Кукина ) и добавление в посылку со статусом светодиода двухбайтного содержимого считанного АЦП, т е каждые 500 ms отправляем 3 байта: (0 или 1) + два байта данные АЦП.
